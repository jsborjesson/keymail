Keymail's documentation

Keymail is a Rails gem that does authentication via email. It was inspired by this blogpost and tries to be a better default than passwords and a simpler alternative to OAuth.

  • Github
  • Example app
  • Example source
  • RubyGems

Keymail does only authentication, it has nothing to do with authorization, if you see the abbreviation 'auth', it always refers to the former.
